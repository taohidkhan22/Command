#Command Line 

## Git repository creation on local computer 
1. mkdir "folder name" <br>
2. git init <br>
3. add, commit<br>
4. git remote add origin "remote repo link"<br>
5. git remote -v(for checking the repo)<br>
6. git branch<br>
7. git push origin main or git push -u origin main
